## Implementation of an Integer Min Heap

#### To run program

```console
user@home:~$ make
g++ -c -std=c++11 main.cpp
g++ -c -std=c++11 IntMinHeap.cpp
g++ -o MinHeap main.o IntMinHeap.o
user@home:~$ ./MinHeap
```

#### Main program commands
Insert: ```Any integer greater than 0```\
Print Heap: ```0```\
Extract Min: ```-1```\
Heap Sort + Print: ```-2```
